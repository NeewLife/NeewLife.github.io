[네트워크] 자주 보는 HTTP 상태 코드들

2024/01/19 카테고리 Network

서버에서의 응답 처리 결과는 응답메세지의 상태 라인에 있는 상태 코드를 보고 파악할 수 있다.

그 중에서 개인적으로 자주 보이는 상태코드를 정리해보았다.

  • 1XX : Information (정보제공)

    • 임시 응답으로 현재 클라이언트의 요청까지는 처리되었으니 계속 진행하라는 뜻이다.

  • 2XX : Success (성공)

    • 클라이언트의 요청이 서버에서 성공적으로 처리되었다는 뜻이다.

  • 3XX : Redirection (리다이렉션)

    • 완전한 처리를 위해서 추가동작이 필요하다는 뜻이다.
      주로 서버의 주소 또는 요청한 URI의 웹 문서가 이동되었으니 그 주소로 다시 시도하라는 의미이다.

  • 4XX : Client Error (클라이언트 에러)

    • 없는 내용을 요청하는 등, 클라이언트의 요청 메세지 내용이 잘못된 경우를 의미한다.

  • 5XX : Server Error (서버 에러)

    • 서버 사정으로 메세지 처리에 문제가 발생하였다는 뜻이다.
      서버의 과부하, DB 처리과정 오류, 서버에서 예외가 발생하는 등의 경우가 있다.

1XX : Information (정보제공)

상태코드 상태 텍스트 서버 측면에서의 의미
100 Continue 계속 진행하라
102 Processing 처리 중이다 → 서버가 처리하는데 오랜시간이 예상되어서 클라이언트에서 타임아웃이 발생하지 않게 할 때 발생함

2XX : Success (성공)

상태코드 상태 텍스트 서버 측면에서의 의미
200 OK 서버가 요청을 성공적으로 처리하였다
201 Created 요청이 처리되어서 새로운 리소스가 생성되었다
202 Accepted 요청은 접수되었지만, 처리가 완료되지 않았다
204 No Content 처리를 성공하였지만, 클라이언트에게 돌려줄 컨텐츠가 없다

3XX : Redirection (리다이렉션)

상태코드 상태 텍스트 서버 측면에서의 의미
301 Moved Permanently 지정한 리소스가 새로운 URI로 이동하였다 → 이동할 곳의 새로운 URI 는 응답 헤더 Location에 기록된다
303 See Other 다른 위치로 요청하라 → 브라우저의 form 요청을 POST로 처리하고 그 결과 화면으로 리다이렉트 시킬 때 자주 보이는 응답코드이다.
307 Temporary Redirect 임시로 리다이렉트 요청이 필요하다 → 요청한 URI 가 없으므로 클라이언트 메소드를 그대로 유지한 채 응답 헤더 Location에 표시된 다른 URI로 요청을 리다이렉트 한다

4XX : Client Error (클라이언트 에러)

상태코드 상태 텍스트 서버 측면에서의 의미
400 Bad Request 요청의 구문이 잘못되었다 → 말 그대로 요청 구문이 잘못됨. 데이터 타입이 안 맞을수도있다.
401 Unauthorized 지정한 리소스에 대한 엑세스 권한이 없다 → 요청자가 누구인지 알 수 없을 때 발생한다. 인증이 필요한 요청인데 실패하거나 제대로 제공이 되지 않은 경우
예) 로그인도 안하고 공유폴더에 접근한 경우
403 Forbidden 지정한 리소스에 대한 엑세스가 금지되었다 → 요청자가 누구인지는 알고 있으나, 401 인증 처리 외의 사유로 리소스에 대한 엑세스가 금지되었음을 의미한다.
예) 로그인은 했는데 공유폴더 접근권한이 없는 경우
404 Not Found 지정한 리소스를 찾을 수 없다

5XX : Server Error (서버 에러)

상태코드 상태 텍스트 서버 측면에서의 의미
500 Internal Server Error 서버에 에러가 발생하였다
501 Not Implemented 요청한 URI의 메서드에 대해 서버가 구현하고 있지 않다
502 Bad Gateway 게이트웨이 또는 프록시 역할을 하는 서버가 그 뒷단의 서버로부터 잘못된 응답을 받았다
503 Service Unavailable 현재 서버에서 서비스를 제공할 수 없다 → 서버의 과부하나 서비스 점검 등 일시적인 상태이다
505 HTTP Version Not Supported 클라이언트가 요청에 사용한 HTTP 버전을 서버가 지원하지 않는다
