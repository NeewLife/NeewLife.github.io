벡터의 종류

  • 숫자형, 정수형, 문자형, 범주형, 논리형 벡터가 존재한다.

  • 범주형 벡터 : 행렬과 비슷한 형태, 연산은 안된다.
    ex) 성별:남자, 여자 / 연령대:청년, 중년, 고령 의 범주를 만들어서 남자청년, 여자중년 등의 값을 만들어내는 벡터

  • 논리형 벡터 : 참(TRUE), 거짓(FALSE)이라는 논리값으로 구성된 데이터. 논리값은 ‘,” 없이 대문자로 쓴다.

  • 벡터 종류 확인

  • 숫자형 벡터

  • 숫자 + 문자 벡터(문자형 벡터로 출력)
  • 범주형 벡터
  • 벡터 만들기
  • data.frame() 에서 한번에 만들기
  • 열 추가하기
  • 데이터의 위치 : 객체 이름[행,열]
  • 복수 데이터 출력
  • 전체 행, 열의 데이터 출력
  • 연속 데이터 출력
  • 필요없는 데이터 제거
  • 특정 변수 출력
