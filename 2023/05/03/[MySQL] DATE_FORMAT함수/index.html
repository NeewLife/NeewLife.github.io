그 외 날짜 관련 함수

DATE(date) : 주어진 ‘date’ 에 해당하는 날짜 반환

MONTH(date) : 주어진 date에서 월에 해당하는 숫자 반환(0~12)

DAY(date) : 주어진 date에서 일자에 해당하는 숫자 반환(0~31)

HOUR(date) : 주어진 date에서 시간에 해당하는 숫자 반환(0~23)

MINUTE(date) : 주어진 date에서 분에 해당하는 숫자 반환(0~59)

SECOND(date) : 주어진 date에서 초에 해당하는 숫자 반환(0~59)

WEEKDAY(date) : 주어진 date에서 요일에 해당하는 숫자 반환(월요일 = 0, 일요일 = 6)

LAST_DAY(date) : 주어진 date에서 해당 월의 마지막 날짜 정보를 반환

SEC_TO_TIME(seconds) : 주어진 seconds 를 기준으로 시간 정보를 반환(HH:MM:SS 형태)

현재 날짜와 관련된 함수

NOW( ), CURRENT_TIMESTAMP( ) : 현재 날짜와 시간 반환

CURDATE( ), CURRENT_DATE( ) : 현재 날짜 반환

CURTIME( ), CURRENT_TIME( ) : 현재 시간 반환
