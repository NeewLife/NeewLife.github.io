산술연산자

  • 숫자형, 정수형 벡터에도 연산이 가능하다

비교연산자

  • a:z : a부터 z까지를 의미
  • seq(a,z,by=2) : a부터 z까지 2씩 증가
  • 3씩 감소
  • rep(x,each=2,times=3) : x를 각각 2번 반복하고, 그 결과를 3번 반복출력

논리 연산자

데이터세트 구조 함수

  • head(,2) : 앞에서 2개 행만 출력
  • tail(,2) : 뒤에서 2개 행만 출력
  • str() : 데이터 전체 구조 및 유형 출력
  • View() : 엑셀 형식으로 출력
  • dim() : 행과 열의 갯수 출력
  • length() : 열의 개수 출력
  • class() : 유형 출력
  • ls() : 벡터 이름을 출력
